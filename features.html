Surface 01

The CLI.

Install globally. Runs in any terminal. Local Whisper voice, auto-exec shell, no browser required. Features interactive diffing, symbol search, and multi-file editing directly from your shell.

Surface 02

Mesh IDE.

A VS Code-style workspace intelligence layer for your browser. Full file tree, structural search, and multi-file editing with live context compression. Perfect for complex refactors.

Surface 03

MCP Server.

Bring Mesh to Claude Desktop or Cursor. Any agent supporting Model Context Protocol can now query your codebase through Mesh's capsule engine. Use Mesh as a local data source for any agent stack.
