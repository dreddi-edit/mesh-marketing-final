Surface 01

Terminal CLI

The CLI stays low-noise and terminal-native. Install globally, run it in any repo, and let Mesh index the workspace before the agent touches files or commands.

Auto-executes shell commands, reads failures, patches files, and retries until the loop is done.
Session capsules preserve context, tool history, and file state across machines.
Local Whisper voice mode keeps voice input offline when you want hands-free commands.

Surface 03

MCP Server

Mesh can run as a local MCP server so other terminal agents use its context engine instead of guessing through raw files. Claude Code, Codex, Cursor, and custom clients can ask Mesh for structured code intelligence.

Expose workspace status, structural search, targeted opens, and exact span recovery as MCP tools.
Keep the user's preferred agent while improving the quality of context it receives.
Tool profiles keep MCP manifests small and avoid noisy tool surfaces.

Surface 04

Gateway API

The Gateway API is the programmable version of Mesh: send a task and workspace reference, receive agent output plus token accounting and source-grounded evidence.

One REST endpoint for workspace Q&A, PR summaries, migration audits, and product workflows.
BYOK model routing across Anthropic, OpenAI, Google Gemini, or Mesh-managed Bedrock.
Stateless calls with explicit token savings and recoverable source references.
