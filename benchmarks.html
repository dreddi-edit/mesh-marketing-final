How we tested

No vibes. No cherry-picking.

Every benchmark runs on the same prompt set across the same five frontier models. Capsules vs raw source. Same temperature (0.2), same max-tokens, same retry policy. Results are reproducible — the run scripts ship in mesh bench.

01

Same models.

claude-sonnet-4.6, gpt-5, llama-3.3-70b, deepseek-v3, gpt-oss-20b. Identical sampling params across both arms.

02

Real workspaces.

247-file fixture mirroring a typical SaaS service: API + workers + db + tests. 18,432 LOC across 12 languages.

03

Reproducible.

Run mesh bench --suite niah,hops,budget,halluc on your machine. Same numbers, every time.
