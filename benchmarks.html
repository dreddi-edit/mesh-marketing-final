A1 - Needle in a Haystack

Can the model find the right file?

A planted constant, SESSION_SECRET, is hidden in one of 10 to 120 files. Raw context runs out before the needle appears. Mesh keeps the full map in budget, then recovers exact spans when the model needs source.

Tested on Sonnet, Opus, GPT-OSS-20B, Gemma 3 4B, and Llama 3.3 70B. Same task, same token budget, same prompt.

Methodology

Part A - live LLM evals

Needle, hard needle, multi-hop, edit precision, and hallucination checks are real inference calls through AWS Bedrock. Same prompt, same token budget, Mesh vs raw side-by-side.

Run: 2026-04-20. Models: Sonnet 4.6, Opus 4.6, GPT-OSS-20B, Gemma 3 4B, Llama 3.3 70B.

Part B - deterministic pipeline

Compression ratios, symbol extraction, capsule determinism, targeted reads, and file-role classification are deterministic CI benchmarks. No LLM calls, no retries, reproducible outputs.

Source: benchmarks/run-all-benchmarks.js. Stack: Node.js 22+, Tree-sitter, Brotli-9, zstd-chunked transport.
