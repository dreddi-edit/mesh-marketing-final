Quickstart

Up in
30s.

No config files. No accounts to create first. One command and Mesh is thinking alongside you.

1
Install
Install globally via npm

Requires Node 18+. Works on macOS, Linux, and WSL on Windows.

$npm i -g @edgarelmo/mesh-agent-cli copy
2
Navigate
Go to your project

Mesh automatically indexes the current directory on startup.

$cd your-project copy
3
Launch
Start Mesh

Mesh launches an interactive REPL. Your codebase is indexed. Start asking.

$mesh copy
4
Optional — Voice
Enable voice mode (macOS)

Installs Whisper via Homebrew for fully local speech recognition.

$mesh doctor voice fix copy
LLM-agnostic. Mesh works with Claude, GPT-4o, Gemini, or any local model via Ollama. Set your key in .env or pass it with --model.
